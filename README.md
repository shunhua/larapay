## 环境要求

* PHP >= 5.6

## 当前支持的接口

### 支付宝接口

* 即时到账（wap,web,app）
* 交易退款接口
* 单笔转账到支付宝

### 微信接口

* 公众号支付
* APP支付
* H5支付
* 扫码支付
* 企业付款到零钱
* 企业付款到银行卡
* 交易退款接口
* 公众号现金红包

### 银联

* 即时到账(wap,web,app)
* 交易退款接口

### Paypal

* APP,网关支付
* 退款
* MassPay(多人转账)

## 安装使用

* 通过composer(推荐)

	```
	composer require "shunhua/larapay"
	```
    
* composer.json

	```
	"require": {
        "shunhua/larapay": "0.*"
    }
	```
    
	然后运行

	```
	composer update shunhua/larapay
	```
